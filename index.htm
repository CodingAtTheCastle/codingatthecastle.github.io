What is Castle Coding?

Castle Coding is an open event at The Castle. It is free for everyone to join, both members of The Castle and outsiders are welcome!

The event is open for all levels. Curious about learning what code is? Want to take you current coding skills to the next level? Kickstarting a fun project and need a place to hangout in while doing it? Castle Coding is the place for you!

The format is "self study" but dont be afraid to ask for some tips and tricks from someone who knows just a little bit more than you.

When?

Every wednesday starting at 17.00 and ending at 20.00

Where?

Location: Spegelsalen (Quiet room)

All levels are welcome! There will be wine!
